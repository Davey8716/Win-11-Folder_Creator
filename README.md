## Usage

Files can either be drag & drop loaded into the app, or created/loaded using the buttons above.  
Supports: `.json`, `.txt`, `.md`

---

### Files
- Default files can be added to folders and subfolders  
- Duplicate default files in the same folder will be ignored  

---

### Templates
User templates saved via the app are stored as `.json`, `.txt`, or `.md`.

A copy is automatically saved to: C:/Users/yourusername/AppData/Local/FolderCreator/


- Right-click the **User Templates** dropdown to open this folder  
- Drag-and-dropped templates are also saved there automatically  

---

### Importing Folder Structures
- Paste folder trees using **Ctrl + V** (plain text format)  
- Drag full folders from your desktop into the tree  
- Files are ignored — only folder structure is imported  

---

### Hotkeys
- `Ctrl + N` → Add Folder  
- `Ctrl + Shift + N` → Add Subfolder  
- `Ctrl + S` → Save Template  
- `Ctrl + O` → Load Template  
- `F2` → Rename  
- `Delete` → Remove selected folder  

---

### Template Tip
Templates can be saved as:
- `.json`
- `.txt`
- `.md`

---

### Tips
- Drag folders to change hierarchy  
- Drag below → create a new parent-level folder  
- Drag onto → make a subfolder  

- Auto-numbering only applies when using **Add Folder / Subfolder buttons**  
- Auto-numbering is disabled for **default templates**  
- Drag-and-drop only changes structure
